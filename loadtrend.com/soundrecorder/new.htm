   What's New in Windows Audio Recorder 4.55

This is the newest version Windows Audio Recorder 4.55 
   Support for Windows Vista  Now  ! 


Windows Audio Recorder 4.55 ---- A 100% Windows Vista-compliant version

This  version  also supports for Windows 95/98/NT/2000/Me/XP

  Built-in Sound Editor ( New ! )
    Now you can edit the recording file or existing audio files by our product. Open the sound and left-click on the sample graph then drag to select the duration you want to edit. It can cut, copy, paste or delete any duration, and support to unlimited undo/redo. Now don't worry about the wrong recording, because our product offer you the chance to correct it.
  Recording management

    This is the update you've been dreaming about! Now Windows Audio Recorder supports file management. You can carry out all your file management through a list box  in the interface of Windows Audio Recorder. You may right-click on recordings in the list and from there you can playback the recordings, convert wav files to MP3 format or delete individual recordings.
  Recording playback
     The new version now supports recording playback. You do not have to start any other sound player program to test recording quality. Just Double-click on recordings in the list to play the  files !
  New mp3 encoder
   With its new powerful Mp3 encoder, Windows Audio Recorder can convert  the recording sample format to Mp3-file format, not only  44100Hz 16Bit Wav files.The MP3 encoder also reduces  conversion time  and improves  MP3 quality.
  Some other changes
    The interface was redesigned with a really easy-to-use user-friendly interface. Some small bugs were eliminated in this version.
