   It does not record anything, all I get is silence (It actual creates a file but without any sound). 

    When you start Windows Audio Recorder for the first time, the program will look for you computer’s available recording sources. The resulting "preference.properties" file will be saved in the path where you installed Windows Audio Recorder. If you delete this properties-file, Windows Audio Recorder will reconfigure your sound card the next time you use the program. Now We advice you to delete the properties-FIlE ,then start Windows Audio Recorder again to reconfig your sound card. (See below : How to configure sound card correctly.)
   How to configure sound card correctly with Windows Audio Recorder ?

    When beginning configuring ,Pleas just obey the prompt .If your sound card support recording function, you will hear test wav sound.

  How long can Windows Audio Recorder record?
   Windows Audio Recorder can consecutively record for as long as you like without interrupt.
  Why the recording quality so bad and mixed terrible noise?
    If the recording has an annoying noise in the background, you can solve this by turning down the recording volume Scrollbar or we suggest you close  the effect of  "bass boost"
  Why the Hotkey doesn't work in DIRECTX mode?(It works fine for Windows NT/2000/XP/2003/VISTA)
     If you want to record some DIRECTX game music, please start game first then switch to Windows Audio Recorder, Press "record" button to record, then switch to game again. Another way is to make a recording schedule for record DIRECTX game music. 
  Will Windows Audio Recorder occupies a lot of memory after a long time recording ? 
   No, With the new techniques called multi-thread, the recording will be saved to the disc once it is captured by Windows Audio Recorder, little bytes will occupy your memory resource, and even the MP3 encoding is real-time processing, no temporary file produced.
